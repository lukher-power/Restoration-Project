The Digital Restoration Archive is designed as a public exhibit, not just a paper substitute. Its main deliverable is a responsive website that explains the project, embeds a Tableau Public dashboard, and gives viewers a guided way to explore early revelations.

The dataset will collect revelation dates, locations, recipients, and short historical notes from the Joseph Smith Papers and related contextual sources. That data will then drive the dashboard views, letting users move between chronology, geography, and theme instead of reading everything in a single linear list.

Deliverable 1

Dataset

A structured source file with the core revelation metadata needed for the visualization.

Deliverable 2

Dashboard

A Tableau Public workbook with map, timeline, and filter controls for the archive.

Deliverable 3

Web Exhibit

A polished site that frames the dashboard with historical interpretation and course-focused analysis.

The site is intentionally narrow in scope so it can be completed well: one dashboard, a few focused interpretive pages, and a consistent visual system that supports the historical material without overwhelming it.
