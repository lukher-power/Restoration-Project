This project depends on a small set of primary and secondary sources that can support both the narrative pages and the Tableau dashboard.

Primary Source

Joseph Smith Papers

Use the manuscript histories, dates, and recipient details to build the core dataset for each revelation.

Contextual Source

Revelations in Context

Use the historical essays to frame the circumstances that surround major sections in the Doctrine and Covenants.

Secondary Scholarship

BYU Religious Studies Center

Cross-reference academic writing on publication history, textual transmission, and early Church development.

Technical Tooling

Tableau Public and GitHub Pages

Publish the dashboard publicly and embed it in the responsive site shell for final presentation.

When the workbook is published, replace the placeholder URL in _config.yml and the homepage will render the live embed automatically.
